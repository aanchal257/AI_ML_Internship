
A Streamlit-based AI chatbot that answers questions about a Samsung Washing Machine manual using **LangChain, OpenAI GPT-4o-mini, and ChromaDB**.

# Features
-  Manual-based Question Answering
-  Retrieval-Augmented Generation (RAG)
-  Interactive Chat Interface
-  OpenAI GPT-4o-mini
-  Chroma Vector Database
