
A simple **Streamlit** web application that demonstrates **K-Means Clustering** for customer segmentation based on **Age** and **Income**.

# Features
- Dataset Preview
- Scatter Plot Visualization
- Min-Max Scaling
- K-Means Clustering
- Cluster Centroids
- Elbow Method
- Clustered Data Display
