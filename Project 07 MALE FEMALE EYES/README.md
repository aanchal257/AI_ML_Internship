
This project is a Streamlit web application that classifies an uploaded eye image as **Male** or **Female** using a Convolutional Neural Network (CNN) built with TensorFlow/Keras.


# Tech Stack

* Python
* TensorFlow / Keras
* Streamlit
* NumPy
* Pillow
